# Zoom AI Interview Prep Showcase

A professional demo showcase website for **Zoom AI Interview Prep**, an AI-powered mock interview application designed to run inside Zoom and help users practice interviews with personalized questions and structured feedback.

## Overview

Zoom AI Interview Prep helps students, job seekers, and career-focused users prepare for real interviews in a familiar Zoom meeting environment. The app uses resume and job description inputs to generate role-specific interview questions and provide AI-supported feedback using the STAR method.

## Features

- Zoom-inspired landing page design
- Hero section with background image, blur overlay, and vignette effect
- Translucent fixed navigation bar
- Demo video section
- Project overview section with matching lighter background
- Animated scroll reveal effects
- Flip-card feature section
- Technology stack tags
- GitHub repository call-to-action
- Responsive design for desktop, tablet, and mobile screens

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Responsive Web Design
- Intersection Observer API
